# 1. Web前端开发之上手伊始

&emsp;&emsp;互联网上有千千万万个网站，每个网站又由许多页面组成。开发大型商用网站是我们的终极目标，我们的起点就是要从做好地一个页面开始。

## 1.1 最简配置：你的小米加步枪

&emsp;&emsp;在开始做你的第一个页面之前，你需要确定你已经准备好了下面清单中所列出的工具：

### a. 一台连接到互联网的台式或者笔记本电脑

&emsp;&emsp;你准备的这台电脑的操作系统可以是微软windows系列中的任意一款（包括win7、win8和win10），也可以是在苹果公司的Macintosh系列电脑上的Mac OS系统（包括OS X、Mac OS 9、Mac OS 8）。当然如果你本人是开源操作系统的发烧友，你也可以选择任意一个类型的linux桌面版操作系统。

&emsp;&emsp;请特别留意：在决定选用任意一款操作系统之前，请你一定首先确认，这些系统不至于太老太旧，以至于它不能为你提供良好的上网体验。

###  b. 一个Web浏览器

&emsp;&emsp;Web浏览器是所有网站的客户端。

&emsp;&emsp;除了你现在看网页正在正用到的浏览器，你还可以为自己即将开始的Web前端学习选择一款专业的浏览器，通常这种给开发者用的浏览器都会提供专业而且强大的开发者工具（developer tools），比如说我们之前提到过的Google Chrome和Mozilla FireFox。

###  c. 一个文本编辑工具

&emsp;&emsp;所有操作系统都提供了自带的文本编辑工具，我们可以直接用来写出自己的网页页面。下面是打开win7默认文本编辑工具的方法：

 ![win7-text-editor](/public/img/html/win7-text-editor.png){:height="400px" width="400px"} 

&emsp;&emsp;我推荐你一款运行在windows系统上的开源文本编辑器：**notepad++**。你只需要打开链接到[notepad++官网](https://notepad-plus-plus.org/)的下载页面下载安装就可以了。

 ![notepad++_download](/public/img/html/notepad++_download.png){:height="400px" width="400px"} 

## 1.2 集成开发环境

### Adobe Dreamweaver

&emsp;&emsp;简称“DW”，中文名称 "梦想编织者"，一款商业收费软件。它是美国Adobe公司开发的集网页制作和管理网站于一身的所见即所得网页编辑器，DW是第一套针对专业网页设计师特别发展的视觉化网页开发工具，利用它可以轻而易举地制作出跨越平台限制和跨越浏览器限制的充满动感的网页。你可以通过[Dreamweaver官方广告](https://helpx.adobe.com/dreamweaver/how-to/what-is-dreamweaver.html)来感受它的强大功能。

# 2. 你的第一个网页

&emsp;&emsp;现在，我们一起动手，准备好在Notepad++里面写一个简单的网页。

## 2.1 创建新文档

### 打开Notepad++

&emsp;&emsp;鼠标双击打开win7桌面上的Notepad++快捷方式：

 ![notepad_plus_plus_icon](/public/img/html/notepad_plus_plus_icon.png)

&emsp;&emsp;如果你的桌面上没有快捷方式，你就在开始菜单里面，选择“所有程序”，导航到Notepad++文件夹里面的快捷方式：

 ![start_menu_open_notepad](/public/img/html/start_menu_open_notepad.png)

&emsp;&emsp;编辑器打开之后，你将会看到一个包含空白文档的Notepad++界面：

 ![start_menu_open_notepad](/public/img/html/notepad++_empty_doc.png){:width="500px" height="400px"} 

## 2.2 保存一个简单的文档

### 复制代码

&emsp;&emsp;在上个小节我们用Google Chrome的开发者工具看到的HTML文档大概是这样的，请你把它复制空白文档里面：

    <html>
    <head><!--此处省略一万个标签--></head>
    <body><!--此处省略两万个标签--></body>
    </html>

---

### 保存文件

&emsp;&emsp;在键盘上按下Ctrl+S,保存这个HTML简单文档。

&emsp;&emsp;HTML文档的文件名后缀必须是“**.html**”，因此我的第一个html文档就应该这样保存：文件名是**hello_web.html**，保存类型选择**All types**。

 ![start_menu_open_notepad](/public/img/html/save_as_html.png){:width="500px" height="400px"} 

---

### 查看效果

&emsp;&emsp;保存成功的文档应该是这个样子：

 ![start_menu_open_notepad](/public/img/html/my_simple_html.png){:width="500px" height="400px"} 


&emsp;&emsp;在Notepad++窗口选择“运行”菜单，Notepad++窗口会出现一个让你选择不同方式打开文档的列表。其中，你可以选择用Firefox、IE、Chrome和Safari这四种浏览器中的任意一个打开当前文件。当然，前提是你的电脑上已经安装好了那个Web浏览器。

 ![open_html_in_chrome](/public/img/html/open_html_in_chrome.png)

&emsp;&emsp;我本地已经安装好了Chrome浏览器，只需要点击“Launch in Chrome”，就可以在Google Chrome里面打开我们刚刚保存页面了，打开之后你会看到：

 ![open_empty_html_doc](/public/img/html/open_empty_html_doc.png)



## 2.3 添加网页内容

&emsp;&emsp;我希望你能跟我一起动手给网页添加内容。

### 网页标题

&emsp;&emsp;首先，我们在\<head\>与\</head\>两者之间增加下面的内容：

          <title>My First Web Page</title>

&emsp;&emsp;F5刷新网页，浏览器窗口的左上方就是展示文档标题的地方：

 ![html-title-demo](/public/img/html/html-title-demo.png)


### 增加文档内容

&emsp;&emsp;然后我们在\<body\>和\</body\>之间增加下面的内容：

    <b>
    Hello, This is my first web page!
    </b>


## 2.4 查看效果

### 检查显示效果

&emsp;&emsp;我们第一个网页的完整代码如下：

    <html>
    <head>
        <title>My First Web Page</title>
    </head>
    <body>
        <b>
        Hello, This is my first web page!
        </b>
    </body>
    </html>

&emsp;&emsp;你需要在Chrome浏览器窗口上F5刷新页面，才能看到页面的最后显示效果：

 ![my_first_web_page_result](/public/img/html/my_first_web_page_result.png)

---

### 在开发者工具中查看网页代码

&emsp;&emsp;F12打开Chrome开发者工具，在Elements面板，查看页面完整代码：

# 3. 关于HTML，不得不说的事儿

&emsp;&emsp;刚才我们创建了一个显示最简单内容的小页面。

&emsp;&emsp;然而这样的网页实在只有用来演示的价值：用户不会从这样的页面上了解到任何新鲜资讯，企业也不会因这个页面而断定你可以承担网站开发的相关职责。如果是第一次看这篇文章，请你明确，这个教程的产生建立在这个假设之上，即：你之前只是听说过HTML这个词，但对于什么是HTML这样的问题，你没有深入思考过，更没有系统学习过。

&emsp;&emsp;制作这个小页面的过程，仅仅是让作为初学者的你，对于什么是网页有留下初步印象。

&emsp;&emsp;HTML的完整名称是**Hyper Text Markup Language**，中文全名是“**超文本标记语言**”，它结合了“超文本”和“标记语言”两种技术、两种特性。
